# tdjrBashMenuScripts
Collection of very simple bash scripts to show menus simplifying some of my common dev tasks with:  
- git  
- lint  
- flow  
- heroku  
...

```
 =============================================================== 
                          Git
 =============================================================== 
   Select:
        Safe Ops                             Push/Pull/Checkout
     ----------------------------------------------------------- 
     a) git add .                    k) git checkout master
     s) git status                   m) git pull origin master
     d) git diff                     M) git push origin master
                                     o) git pull origin
                                     O) git push origin
                c) show Code Check menu
                h) show Heroku menu
                q) quit (or any other key)
     ----------------------------------------------------------- 
 
```

```
 =============================================================== 
                          Code Check
 =============================================================== 
   Select:
        JavaScript                       Ruby/Rails
     ------------------------------------------------------ 
     f) run flow type checking
     l) run lint checks               r) run rubocop checks
     t) run unit tests                t) run unit tests
     a) all of the above              A) all of the above
                g) show Git menu
                q) quit (or any other key)
     ------------------------------------------------------ 
```
